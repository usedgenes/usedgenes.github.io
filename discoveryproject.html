Discovery Project

Inspiration/Idea

In high school, I had learned to design my own PCB to use as a flight computer on a rocket. It was used to control various mechanisms like a thrust vectoring control system. Then, here at Georgia Tech, I brought that experience with me to the controls subgroup of the competition team of Ramblin' Rocketry Club, designing another flight computer to control an airbrake system, and replacing the initial plan of soldering together breakout boards. 

Project Progress

I finished all the online design, both the schematic and the PCB layout. The only thing I have left to do is to actually order it from a PCB manufacturer and get it delivered. Then, I'll have to solder a few components on, but overall, the project is 90% done.

ECE Skills Gained

- Circuit analysis/design
- Reading datasheets, component selection
- Power design
- EDA software knowledge
- Signal integrity (decoupling capacitors, power/ground planes)
- DRC checking

Success/Failures, Final Thoughts

Overall, this project was a success. I learned a lot of things about PCB design, and specifically things regarding how to design with STM32s, through things like how to use inductors to filter analog voltage and also using external crystals. I didn't really have any failures since everything so far has been done online, and I'll find out about any mistakes once I get the PCB and connect power to it for the first time. (No magic smoke, fingers crossed). In addition, I'll also be writing the code to actually control the airbrake using this PCB, along with the sensor and STM32 drivers.

Overall design:
STM32F405 MCU
USB-C + SWD ports
SD card module + FLASH memory
Altimeter, IMU, Magnetometer

Closing remarks: I've spent around 15-20 hours on this project doing everything from reading datasheets. It was a lot of fun designing everything how I wanted, and doing the PCB routing was really engaging, as it kind of felt like solving a puzzle trying to figure out the optimal spacing and way to connect things. The project isn't too related to the threads I'm interested in (Circuit Tech + Signal Processing), but it has made me a lot more interested in electrical engineering overall.
