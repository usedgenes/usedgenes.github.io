FPAA Standard Cell Development
(Oct. 2025 - Present)

As a part of the Integrated Computational Electronics Lab at Georgia Tech, I've been familiarizing myself with VLSI design, using tools like Xschem for schematic creation and Magic VLSI for layout design, specifically targeting the Sky-130nm process node. I'm actively creating standard cells for a library to use to design Field Programmable Analog Arrays that the lab is working on. The goal is to open-source our designs and help analog circuits more widespread, as there is very little in the way of analog EDA software currently.

Airbrake + Carbon Fiber Rocket
(Dec. 2023 - March 2025)

I started this project with the goal of creating an airbrake mechanism to actively control the speed of a model rocket in flight. The idea was to use an Arduino flight computer to measure the live velocity, and then use a PID control loop to deploy flaps that would increase/decrease the rocket's coefficient of drag accordingly. After iterating through different designs, I was able to get the mechanism to achieve landings within 1 foot of a pre-determined target altitude. 
Along the way, I also started a side-project manufacturing my own custom carbon-fiber tubes, as I wanted to increase the structural strength along with the reusability of my rocket tubes. After 10+ iterations, I was able to successfully achieve both of those goals, and I reduced replacement costs by 50% compared to traditional paper airframes used to build rockets.
https://github.com/usedgenes/Airbrake

PCB Flight Computer
(June 2024 – Dec. 2024)

This project was an extension of the airbrake project above. After using my Arduino flight computer for a while, I was unsatisfied with its size and complexity, especially the wires that got in the way of everything. So, I started learning how to design my own PCBs and got them manufactured. This final iteration of my flight computer used an ESP32, and uses two altimeters, two IMUS, and one magnetometer for sensor data. It also has an SD card module for data logging, a buzzer and LEDs for user feedback, and 5 pins to connect servos to. 
https://github.com/usedgenes/Flight-Computer-PCB

iOS Rocketry Assistant App
(June 2024 – Dec. 2024)

This project was also an extension of the above project, the PCB flight computer. Since ESP32 has Bluetooth capabilities, I decided to take advantage of that and coded my own iOS app to help with control and debugging. Without any prior experience with app development, I had to learn Swift from scratch, though it was a very fun learning experience overall. The app included the following features: visualizing sensor data through graphs, changing PID coefficients and controlling servos live, and changing current rocket state for software-in-the-loop testing.
https://github.com/usedgenes/RocketController
